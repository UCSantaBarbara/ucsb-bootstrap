Header

The header is a complex UI component, containing a number of subcomponents. Source code for a full example is provided below.

Desktop View

Mobile View

Subcomponents

The header includes the following components:

University Lock-up

A link to the main UCSB website. Appears in the top-left corner in the desktop view and disappears in the mobile view.

Auxiliary Navigation

Secondary navigation (e.g. "Give Now", "Quick Links", etc.) that appears in the top-right corner in the desktop view and shifts to the bottom part of the menu in the mobile view.

Website Lock-up

Logo of the current website. Appears as the major focal point of the header in both the desktop and mobile view.

Main Navigation

Main navigation for the website. Appears as a horizontal list in the desktop view and a vertical list in the mobile view.
