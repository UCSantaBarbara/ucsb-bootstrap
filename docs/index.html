UCSB Bootstrap Theme

Header

The header is a complex UI component, containing the website's lock-up, navigation, and search. A full example is provided below, but we'll go through building up the component step-by-step:

Wrapping Element

First, we'll start with a wrapping <div> that has the .header class: 

<div class="header">
  <-- Header Contents -->
</div>

Lock-up

Inside the .header <div>, we add the "lock-up", or logo, for our website: 

<div class="lockup">
  <div class="container">
    <h1 class="lockup-title">
      <a href="https://your-website-address.ucsb.edu/">Your Website Name</a>
    </h1>
  </div>
</div>

This will simply add a lock-up to the website, with no header background:

You'll notice there are no images in the HTML: the link will actually be swapped out with the image of our lock-up using CSS.

Eyebrow

Next, we'll add the "eyebrow", which is a thin blue bar at the top of the website that contains two elements: the UCSB lock-up and the mobile navigation toggle control: 

<div class="eyebrow">
  <div class="container">

    <a class="ucsb-lockup" href="https://ucsb.edu/">University of California, Santa Barbara</a>

    <button class="navbar-toggler" type="button" data-toggle="collapse" data-target="#nav-wrapper" aria-controls="nav-wrapper" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>

  </div>
</div>

In the desktop view, the .ucsb-lockup <div> will appear in the top-left corner:

But in the mobile view, the .ucsb-lockup <div> will disappear and the website lock-up will shift into its place. Along with that, the mobile navigation toggle control will also appear:

Header Navigation Wrapper

Next, we'll add a wrapping .header-nav <div> to contain the rest of our navigation elements: 

<div class="header">

  <!-- Header Lock-up -->
  <div class="lockup"><!-- ... --></div>

  <!-- Eyebrow -->
  <div class="eyebrow"><!-- ... --></div>

  <!-- Header Navigation Wrapper -->
  <div id="nav-wrapper" class="header-nav collapse">
    <!-- This is where we'll put the rest of our navigation elements -->
  </div>

</div>

As you can see, this .header-nav <div> has an id of #nav-wrapper and a class of .collapse. This allows us to control the visibility of the rest of our navigation elements using the mobile navigation toggle control we added above.
